 Using Web Audio API to create a sound that varies in frequency.
 
 * One oscillator produces sound. You set the base frequency with mouse Y.
 * Another oscillator (LFO) controls the how fast the frequency of the first oscillator should change over time. You set the LFO frequency with mouse X.
 * A gain node controls the amplitude of the LFO. Set with the slider.
